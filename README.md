## üíª Running Locally

1. **Clone the Source**
   ```bash
   git clone https://github.com/Gamingllc/Nicheyy.git
   cd Nicheyy
   ```

2. **Fuel Up**
   ```bash
   npm install
   ```

3. **Ignite**
   Create a `.env.local` and add your Secret Key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

4. **Launch**
   ```bash
   npm run dev
   ```

## üì¶ Zero-Config Deployment
This app is ready for instant deployment to **Vercel**. 

1. Push this repo to GitHub.

3. Add `VITE_GEMINI_API_KEY` to your environment variables.
4. Ship it.
